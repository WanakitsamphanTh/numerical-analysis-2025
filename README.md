# Numerical Analysis 2025
This repository contains my homework assignments for the Numerical Analysis course (数値解析) during the first semester of 2025. (Grade 91/100)

## Language and Tools
I was instructed to use C and gnuplot for this course. However, I used Python for homework 2 due to issues with my IDE.
All programs ran on Windows 10.

## Contents
**Homework 1** 
- IEE754 
- Integer overflow
- Rouding error, truncation error, and error propagation

**Homework 2** 
- Numerical derivation and integral
- Numerical improper integral (as independent project)
- Error bound calculation in numerical derivation and integral

**Homework 3** 
- Lagrange interpolation
- Least Square Method
- Solving non-linear problems with bisection method and Newton's method
- Comparing Spline's interpolation to Lagrange interpolation (as independent project)

**Homework 4** 
- Solving linear equations with Gaussian elimination and Jacobian iteration method
- Solving first order differential equations with Euler's method and Runge-Kutta methods
- Step size and errors in Euler's method and Runge-Kutta methods
- Solving first order differential equations with Heun's method (as independent project)

**Homework 5**
- Monte-Carlo method
- Solving 2D Laplace's equation
- Calculating the error bound in Monte-Carlo method for value approximation of Pi (as independent project & discussion)
